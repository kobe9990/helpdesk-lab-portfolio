# 🏠 Home Lab Setup

## Overview
A virtualized home lab environment built to simulate a small office IT setup.
Used for hands-on practice with hardware, OS configuration, and networking concepts.

## Environment
- **Host Machine:** Mac
- **Virtualization Software:** VirtualBox
- **VMs Running:** Windows 11, Ubuntu 22.04

## Lab Goals
- Practice OS installation and configuration
- Simulate help desk troubleshooting scenarios
- Document real setup steps as if onboarding a new machine
