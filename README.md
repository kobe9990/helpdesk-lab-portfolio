
# Home Lab 1 

A virtualized home lab environment built on macOS using UTM, featuring Windows 11 and Ubuntu VMs. Designed to gain hands-on experience with operating system installation, configuration, network setup, and troubleshooting

## 📁 What's Inside

| Section | Description |
|---|---|
| 🏠 Home Lab Setup | VirtualBox environment with Windows 11 and Ubuntu VMs |
| 🛠️ Troubleshooting Guide | Common Windows and Linux errors with step-by-step fixes |
| 🎫 Ticket Log | 25+ simulated help desk tickets with resolutions |

## 🧰 Tools Used
- VirtualBox (virtualization)
- Windows 11 & Ubuntu (operating systems)
- draw.io (network diagrams)
- Markdown (documentation)

## 🎯 Goal
To demonstrate practical IT support skills including OS troubleshooting,
hardware/software configuration, and help desk documentation.
